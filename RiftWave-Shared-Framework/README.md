# RiftWave Shared Framework

This repo package contains the shared framework used by:
- WaveForm / RiftSynth Lite
- WaveForm Pro
- RiftSynth Pro

Included:
- `Shared/` common analysis, synth, preset, playback, export, and UI shells
- `Assets/SFX/` shared UI SFX placeholders
- `cmake/` common CMake helpers
- docs/scripts/release scaffolding needed for integration and ship-gate work

Use this as the future shared-core repo or submodule source if you decide to split the plugin trio into separate public repositories.
