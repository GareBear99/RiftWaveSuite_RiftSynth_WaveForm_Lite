# Host Validation Playbook V8

## Primary hosts
- Logic Pro: AU scan, recall, offline bounce
- Ableton Live: VST3 insert, automation, transport sync
- Reaper: AU/VST3 deep validation, rapid open/close, buffer changes
- FL Studio: VST3 UI open/close, MIDI drag, preset recall

## Mandatory cases
1. fresh scan
2. insert on empty track
3. save project and reload
4. automate 3 parameters rapidly
5. change sample rate and reopen
6. run offline bounce
7. drag MIDI from RiftSynth Pro in a supported host
8. toggle beauty mode and UI SFX while audio plays
