# Release Gate

Do not ship until every statement below is true:

1. All three plugins compile cleanly from the same workspace
2. Bundle IDs and plugin codes are unique and conflict-safe
3. Save/load recall is correct in multiple DAWs
4. WaveForm Pro visuals remain smooth under real project load
5. RiftSynth Pro timing is stable on transport start/stop and looped playback
6. Lite feels immediately impressive within 10 seconds
7. Presets and seed-variation hooks are usable and non-buggy
8. No reproducible crash remains open
9. Screenshots, GIFs, changelog, and install docs exist
10. macOS signing/notarization plan is ready
